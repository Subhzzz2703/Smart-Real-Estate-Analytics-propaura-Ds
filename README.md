## 🎯 About PropAura

PropAura is a complete AI-powered Real Estate Intelligence Platform
built using Data Science and Machine Learning to analyse, predict
and visualise Tamil Nadu property market trends across
**12 cities** and **158 localities**.

---

## 🏙️ Cities Covered

Chennai · Coimbatore · Madurai · Trichy · Salem · Tirunelveli
· Erode · Vellore · Thanjavur · Tiruppur · Hosur · Nagercoil
